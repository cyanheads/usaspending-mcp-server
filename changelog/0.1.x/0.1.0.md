# 0.1.0 — 2026-05-25

## Added

- **`usaspending_search_awards`** — search federal awards by keyword, recipient, agency, award type, NAICS, PSC, location, and date range
- **`usaspending_get_award`** — fetch full award details by generated ID, including recipient hash, parent IDV, subaward count, and funding account linkages
- **`usaspending_get_award_transactions`** — list obligation history and modifications on an award
- **`usaspending_get_award_subawards`** — list subaward contracts and grants under a prime award
- **`usaspending_search_recipients`** — search organizations receiving federal funds by name or UEI
- **`usaspending_get_recipient`** — fetch recipient profile with address, business types, parent org, and award totals
- **`usaspending_get_agency`** — fetch agency overview; accepts both `toptier_code` and `agency_slug` to avoid intermediate lookups
- **`usaspending_spending_by_geography`** — aggregate spending by state, county, or congressional district with per-capita output
- **`usaspending_spending_by_category`** — aggregate spending by NAICS, PSC, agency, CFDA, or recipient using a `category` enum (consolidates 14 sub-routes)
- **`usaspending_spending_over_time`** — aggregate spending by fiscal year, quarter, or month
- **`usaspending_disaster_spending`** — disaster and emergency supplemental spending via `dimension` + `spending_type` enums (consolidates 9+ endpoints)
- **`usaspending_get_federal_account`** — fetch federal account budget data: obligations, outlays, program activities, object class breakdown
- **`usaspending_list_agencies`** — list all top-tier agencies with toptier codes, slugs, and fiscal year totals
- **`usaspending_autocomplete`** — discover valid NAICS, PSC, CFDA, agency, and recipient codes by description
- **`USASpendingService`** — single HTTP client wrapping `api.usaspending.gov/api/v2/` with retry, timeout, and parse-failure classification
- **Server config** — `USASPENDING_BASE_URL` and `USASPENDING_TIMEOUT_MS` env vars validated via Zod at startup
