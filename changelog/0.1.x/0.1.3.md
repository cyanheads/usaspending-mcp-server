# 0.1.3 — 2026-05-26

## Changed

- **`package.json` scripts** migrated from `tsx` to `bun run` (`build`, `rebuild`, `clean`, `devcheck`, `tree`, `list-skills`, `lint:mcp`, `lint:packaging`, `changelog:build`, `changelog:check`); `start:stdio` and `start:http` now use `bun ./dist/index.js` instead of `node`; `test` uses `bunx vitest run`; removed bare `start` alias
- **`package.json` metadata** — added `author`, `bugs`, `homepage`, and `funding` fields; removed unused `tsx` devDependency
- **README badges** — added Docker (ghcr.io), MCP SDK, and npm badges to the first badge row; added Install in Claude Desktop, Install in Cursor, and Install in VS Code one-click install buttons
- **Dockerfile** — build stage uses `bun run build` (via script); added `org.opencontainers.image.source` OCI label; corrected `org.opencontainers.image.title` to scoped package name
- **Docker image** — multi-arch (`linux/amd64`, `linux/arm64`) image published to `ghcr.io/cyanheads/usaspending-mcp-server`
- **`.github/FUNDING.yml`** — added GitHub Sponsors and Buy Me a Coffee funding configuration
