# 0.2.1 — 2026-06-12

## Added

- **`truncated` / `shown` / `cap` enrichment** on `usaspending_autocomplete`, `usaspending_get_idv_awards`, and `usaspending_search_recipients` — endpoints that return no total count now disclose a full page via `ctx.enrich.truncated()` with guidance to raise the limit or page forward.
- **Dockerfile `HEALTHCHECK`** — bun-native `fetch` against `/healthz` (the slim runtime image ships no curl/wget).
- **`APP_VERSION` build arg** wired to the `org.opencontainers.image.version` OCI label.

## Changed

- **Enrichment total fields renamed to `totalCount`** and emitted via `ctx.enrich.total()` across `usaspending_search_awards`, `usaspending_search_federal_accounts`, `usaspending_spending_by_category`, `usaspending_disaster_spending`, `usaspending_get_award_subawards`, and `usaspending_get_award_transactions` — replaces the per-tool `total` / `result_total` / `subaward_total` / `transaction_total` keys with the framework's canonical signal.
- **`createApp()`** sets explicit `name` and `title` to `usaspending-mcp-server` (display identity is the hyphenated repo name on every surface).
- **`.codex-plugin/plugin.json`** `interface.displayName` → `usaspending-mcp-server` (was the npm-scoped name).
- **`bundle` script** runs `scripts/clean-mcpb.ts` after `mcpb pack` to prune dev-dependency and agent-doc content the pack step cannot exclude.
- **`.mcpbignore`** patterns root-anchored (`/.claude/`, `/Dockerfile`, …); added `/.agents/` and `/skills/`.
- **Vendored skills** re-synced to framework 0.10.6; added the `techniques` skill.
- **`CLAUDE.md`** dropped the one-time scaffolding "First Session" block.

## Dependency bumps

- `@cyanheads/mcp-ts-core` `^0.9.21` → `^0.10.6`
- `@types/node` `25.9.2` → `25.9.3`
